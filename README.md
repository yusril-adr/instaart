## Build and Test
- Be sure your computer is already installed node.js and npm.
- Install the module with command : 
```bash 
  npm i
```
- Run developer mode by command :
```bash 
  npm run dev
```
- Or if you want to build production mode you can run this command :
```bash 
  npm run build
```
